# DevOps Playground & Knowledge Hub

A personal repository for DevOps challenges, hands-on labs, notes, cheat sheets, and experiments.

## What's inside?

This repo is my living notebook for everything DevOps — from basic concepts to advanced tooling.

### Main Topics Covered
- Linux & Scripting (Bash, Python, Go...)
- Containers & Orchestration (Docker · Kubernetes · Podman)
- CI/CD Pipelines (GitHub Actions · GitLab CI · Jenkins · ArgoCD)
- Infrastructure as Code (Terraform · Pulumi · Crossplane · Ansible)
- Cloud Platforms (AWS · GCP · Azure · OpenShift)
- Monitoring & Observability (Prometheus · Grafana · Loki · ELK · SigNoz · OpenTelemetry)
- Service Mesh & Networking (Istio · Linkerd · Cilium)
- Security & DevSecOps (Trivy · Falco · OPA · Kyverno)
- GitOps workflows & best practices
