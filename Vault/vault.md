Hashicorp Vault
---------------

Secret management tool used for storing passwords, API Keys and Tokens and Certificates.
Vault secrets are encrypted in transit and in storage.

  Users: Humans and system users are granted access control (RBAC)
  Security should be layered to add extra protection and resiliency.

Types
1. Self Managed - Deployable by K8s
2. HCP Vault Dedication - Enterprise level - Cloud based and managed

Vault provides -  Dynamic Credential Generation, Automated Secret Rotation, Detailed Audit Logging.
